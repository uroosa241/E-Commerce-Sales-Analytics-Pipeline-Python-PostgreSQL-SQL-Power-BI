# E-Commerce Sales Analytics Pipeline | Python, PostgreSQL, SQL, Power BI

## Project Overview

Designed and developed an end-to-end sales analytics pipeline leveraging Python, PostgreSQL, SQL, and Power BI to transform raw transactional data into actionable business insights. The project involved data cleaning, relational database modeling, advanced SQL analysis, and interactive dashboard development.

## Objectives

* Build a scalable analytics workflow from raw CSV files to business-ready dashboards.
* Ensure data quality through preprocessing and validation.
* Implement a normalized relational database schema with enforced referential integrity.
* Perform exploratory and business-driven analysis using SQL.
* Deliver interactive visualizations for executive-level reporting.

## Tech Stack

**Programming:** Python (Pandas, NumPy)

**Database:** PostgreSQL

**Query Language:** SQL

**Visualization:** Power BI


## Project Workflow

### 1. Data Cleaning and Preprocessing (Python)

* Handled missing values and inconsistent records.
* Standardized data types and date formats.
* Identified and corrected invalid Product IDs and Customer IDs.
* Generated cleaned datasets for downstream analysis.
*  Performed Statistical Analysis

### 2. Database Design (PostgreSQL)

Created normalized tables:

* Customers
* Products
* Orders
* Employees
* Campaigns

Implemented:

* Primary Keys
* Foreign Keys
* One-to-Many Relationships
* Referential Integrity Checks

### 3. SQL Analytics

Performed analysis including:

* Customer Lifetime Value (CLV)
* Average Order Value (AOV)
* Monthly Revenue Trends
* Product Performance Analysis
* Customer Segmentation
* Repeat Purchase Behavior
* Churn Identification
* Running Revenue Calculations
* Ranking and Window Functions

### 4. Power BI Dashboard Development

Developed interactive dashboards featuring:

* Revenue KPIs
* Monthly Sales Trends
* Top Performing Products
* Customer Insights
* Campaign Effectiveness
* Dynamic Filters and Drill-through Reports

## Key Achievements

✔ Cleaned and validated 10,000+ transactional records

✔ Built a normalized PostgreSQL database with enforced relationships

✔ Solved foreign key inconsistencies and maintained data integrity

✔ Developed advanced SQL queries using joins, CTEs, aggregations, and window functions

✔ Designed executive-ready Power BI dashboards for business stakeholders
